Connect Controller

Talks to your Arduino directly over USB using the Web Serial API (9600 baud, expects lines like CARD:FAD0D505). Works in Chrome or Edge on desktop only, and the page must be served from http://localhost or https:// — not opened by double-clicking the file.

Simulate an incoming tap

Looks up the card ID against the ID learner list and lights the matching lamp — the same path a real NFC tap would take.

Raw serial monitor

Shows exactly what's arriving from the Arduino, unfiltered. Tap a card and watch here — if nothing appears, the problem is between the Arduino and the browser, not in this app.
